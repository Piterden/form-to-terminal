# Form-to-Terminal

Fill out Google Forms on the command line.

## Installation

```bash
npm install -g form-to-terminal
```

## Usage

```bash
$ ftt [GOOGLE_FORM_ID | GOOGLE_FORM_URL]
```

## Limitations

Currently only supports text inputs. One thing at a time!
